Programació

S'introdueixen els fonaments de la programació utilitzant diversos llenguatges com Java o Python. Es fa èmfasi en les estructures de dades, algorismes, la programació orientada a objectes i el desenvolupament de petits projectes.

En aquest mòdul crec que aprendré a programar amb Java i PHP, practicant especialment la programació orientada a objectes.

Espere que no estiga centrat només en aspectes matemàtics i que siga variat i aplicat.
